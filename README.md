## GET PROXIES
Get Proxies Live With Python >=3.9

### Instalation
```python
pkg install python
pip install requests requests[socks]
git clone https://github.com/g4rzk/get_proxies
```

### Run
```python
cd $HOME/get_proxies
python run.py -o <OUTPUT>
```

### Help command
`python run.py -h`

### Module
`requests` and `requests[socks]`
